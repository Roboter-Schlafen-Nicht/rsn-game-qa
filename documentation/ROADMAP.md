## Current Phase

Autonomous AI-driven game QA for browser-based games. The platform
captures gameplay via headless browsers, detects anomalies using
computer vision oracles, and trains RL agents to explore game states.

**Supported games:** Breakout71, Hextris, Shapez
**Stack:** Python 3.12, Gymnasium, Stable-Baselines3, YOLO, Selenium
